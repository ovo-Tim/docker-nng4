# docker-nng4
Docker image for [NNG4(Natural Number Game)](https://github.com/leanprover-community/NNG4).

Docker Hub: [ovotim/nng4](https://hub.docker.com/r/ovotim/nng4)
Run: `docker run -d -p 3000:3000 ovotim/nng4:latest`
